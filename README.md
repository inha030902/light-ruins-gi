# 빛의 잔향

고대 유적을 탐험하며 세 개의 빛 유물을 회수하고, 색이 맞는 받침대에 배치해 중앙 제단을 복원하는 3D 웹 어드벤처 퍼즐 게임입니다.
총 3개의 스테이지로 구성되며, 스테이지가 올라갈수록 빛 함정의 수와 속도가 증가합니다.

## 실행

정적 웹 프로젝트입니다. 로컬에서는 간단한 정적 서버로 실행할 수 있습니다.

```bash
python -m http.server 5173
```

브라우저에서 `http://localhost:5173`을 열면 됩니다.

## 조작

- `WASD`: 이동
- `마우스`: 시점 조작
- `Space`: 점프
- `E`: 유물 들기, 놓기, 회수
- `V`: 1인칭/3인칭 전환
- `E`: 흰 별 기둥 GI 장치 활성화
- `P`: DDGI probe grid 시각화
- `H`: 리포트 캡쳐용 HUD 숨김/표시

## 게임 장치

- 통로의 붉은 빛 함정에 닿으면 시작 지점으로 돌아갑니다.
- 함정에 닿으면 GI가 꺼지고, 들고 있던 유물과 받침대에 놓은 유물이 모두 초기 위치로 돌아갑니다.
- 처음에는 유물과 받침대가 흰색으로 보이며, 붉은 함정을 지나 안쪽의 흰 별 기둥 GI 장치를 켜야 색이 드러납니다.
- 각 스테이지마다 GI 장치를 다시 켜고 세 유물을 모두 맞게 배치해야 합니다.
- Stage 1과 Stage 2 클리어 후에는 뒤쪽 문이 열리고, 문 너머 별도 공간의 빛의 다리를 점프로 건너야 다음 스테이지가 시작됩니다.
- Stage 2 이후의 빛의 다리는 발판이 더 좁고 간격이 불규칙해집니다.
- Stage 3을 클리어하면 마지막 문이 열립니다.

## 구현 포인트

- Three.js 기반 웹 3D 게임
- 1인칭/3인칭 카메라 전환
- 유물 운반 및 받침대 배치 퍼즐
- 태양, 달, 새싹 이름에 맞는 유물 실루엣
- 스테이지 전환용 점프 다리와 낙하 실패 판정
- 성공, 실패, GI 기동, 점프 효과음
- 진행 상태 HUD와 리포트 캡쳐용 HUD 숨김 기능
- 게임 진행 단계마다 다음 행동을 알려주는 안내 프롬프트
- 로블록스풍 블록 캐릭터와 빛 함정 장치
- 단순화된 DDGI 스타일 probe grid 기반 간접광
- 리포트 캡쳐용 흰 별 기둥 GI 장치 및 probe debug view
